A fast way to allow .ogv file uploads to the media library.  

## Description ##
This plugin enables .ogv file uploads to the media library. WordPress doesn't allow .ogv movie file type uploads unless you enable this plugin.

## Installation ##
1) Download plugin .zip file.
2) Login to WordPress dashboard
3) Plugins >> Add New >> Upload .zip File
3) Activate Plugin

## Help ##
There's a screencast video on how to activate this plugin on the [plugin Author page](https://www.slibdesign.com/allow-ogv-file-uploads-wordpress-plugin/)

## Information ##
This pugin works as soon as activate it :)
